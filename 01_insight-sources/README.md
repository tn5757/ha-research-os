# 01_insight-sources

insightsを生むすべてのインプット（思考の素材）を蓄積する場所。

## 役割

- 「読む・観る・議論する」の記録を構造的に保存する
- `02_insights/` での蒸留の原材料となる
- Claudeは文献レビュー・考察の生成時にこのフォルダを参照する

## サブフォルダ

| フォルダ | 内容 |
|---|---|
| `literature/` | 先行研究（論文・書籍の要約とメモ） |
| `field-notes/` | フィールドノート（授業観察等） |
| `meeting-logs/` | MTGログ（ブレスト、研究会議の記録） |

## ここに置かないもの

- 引用文献の書誌情報 → `03_papers/references/`
- 自分が集めた一次データ（アンケート結果等） → `03_papers/*/data/`
