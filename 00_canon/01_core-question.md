## 核心的な問い

> **日本の医療者教育において、ヘルス・アドボカシーの能力はどのように育まれうるか？**

## この問いが生まれた現場の困難

1. **HA教育プログラムが日本にほぼ存在しない** — 小児科専門医の到達目標に「子どもの代弁者」があるのに、それを教えるプログラムがない
2. **HAは「教えにくい」とされている** — 定義が多様、必要な能力が複合的、観察機会が少ない
3. **知識の指導だけでは不十分** — HAには社会的文脈があり、単一の介入では実践に至らない
4. **評価方法も確立していない** — 何をもって「HAができるようになった」と言えるのか不明

## 問いの階層構造

```
【上位の問い】
日本の医療者教育において、HAの能力はどのように育まれうるか？
  │
  ├─【中位の問い①】教育プログラムの設計
  │   短期プログラム（CHAT等）は、HAの認識・行動変容にどう影響するか？
  │   長期的・段階的なHA教育はどのように設計されるべきか？
  │
  ├─【中位の問い②】HA実践の促進・阻害要因
  │   ミクロ→メゾ→マクロレベルのHA実践を支える条件は何か？
  │   実践障壁をどのように取り除けるか？
  │
  └─【中位の問い③】評価と質保証
      HA能力の変容をどう捉え、評価すべきか？
      経験学習サイクルとHA能力開発はどう関係するか？
```
