# Methodology: 研究方法

## 1. CHAT（Child Advocacy Training）プログラムの概要

### 1.1 基本情報

| 項目 | 内容 |
|---|---|
| **開始年度** | 2020年度 |
| **対象** | 全国の小児科専攻医（1〜3年目） |
| **実施年度** | 2022年度 |
| **参加者数** | 37名（先着順） |
| **全回参加者数** | 27名 |
| **開催形式** | PBL（Problem-based learning）形式のワークショップ |
| **実施方法** | Web会議システム使用 |
| **開催期間** | 11月〜2月（4ヶ月間） |
| **開催頻度** | 月1回、合計4回 |
| **各回の時間** | 2時間 |

### 1.2 運営体制

**ファシリテーター構成**:
- **人数**: 11名
- **専門性**: HAの概念を熟知し、かつ実践を行っている全国の小児科医
- **役割**: 講義とグループディスカッションの運営

**グループ編成**:
- 各グループ5〜7名の小グループ

### 1.3 プログラム構成

CHATは講義と小グループ演習を組み合わせたPBL形式で構成される。以下に各回の詳細を示す。

#### 第1回：バイオサイコソーシャルな視点の習得

| 要素 | 内容 |
|---|---|
| **到達目標** | ・子どもと家族をバイオサイコソーシャルな視点で評価できる<br>・医学的な情報、患者個人の情報、家族の情報、社会的な情報の４分割表を使用できる<br>・子ども・家族の視点で子どもに影響を与えている課題を抽出できる |
| **事前課題** | なし |
| **プログラム内容** | 1. アイスブレイク<br>2. グループワークの概要および模擬症例１についての説明<br>3. グループワーク（模擬症例1の4分割表作成および課題抽出）<br>4. 全体共有（各グループの成果物発表） |
| **模擬症例1** | 発達特性があり、親子関係などの社会的要因に様々な課題がある11歳女児 |

#### 第2回：課題抽出と解決方法の検討

| 要素 | 内容 |
|---|---|
| **到達目標** | ・子ども・家族の視点で、子どもに影響を与えている課題を抽出できる<br>・課題に対する解決方法についてグループ内で検討できる |
| **事前課題** | グループメンバーとHAに関する共通点を討論し、グループ名とグループを表すイラストを作成する |
| **プログラム内容** | 1. グループワークの概要および模擬症例2についての説明<br>2. グループワーク（模擬症例2の4分割表作成および課題抽出）<br>3. 全体共有（各グループの成果物発表） |
| **模擬症例2** | 染色体異常（両大血管右室起始症、気管軟化症、嚥下障害、便秘症、精神運動発達遅滞）がある7歳男児 |

#### 第3回：多職種連携の視点

| 要素 | 内容 |
|---|---|
| **到達目標** | ・ステイクホルダーの視点（院内外）に気づき、協働することの意味が分かる |
| **事前課題** | 第２回ワークショップで討議した課題・解決方法について、各参加者の自施設で多職種に聞き取り調査を実施しまとめる |
| **プログラム内容** | 1. 事前課題の情報（多職種連携の視点や活用できるリソース等）をグループ内で共有<br>　※ゲストとして参加した多職種（社会福祉士と訪問看護師など）もグループワークに参加<br>2. 全体共有<br>3. 社会福祉士と訪問看護師の講義（「自分ならこの症例にどのように関わるか」）<br>4. グループワーク（症例の支援のあり方について再度考察） |

#### 第4回：政策提言への視点

| 要素 | 内容 |
|---|---|
| **到達目標** | ・参加者が実際に経験した症例を使用し、課題の抽出をし、解決方法について検討できる<br>・政策提言につながるような活動を知る |
| **事前課題** | 自分が経験した症例に関して、実践できそうなHAを検討する |
| **プログラム内容** | 1. 自分が経験した症例に関するHAをグループ内で共有<br>2. 政策につなげる方法について講義<br>3. 社会の仕組みとして実際にあったら良いと思う支援について、グループ内で討議 |

### 1.4 教育方法の特徴

- **段階的学習**: ミクロレベル（個別患者）から、メゾ・マクロレベル（政策）への視点拡張
- **実践的アプローチ**: 模擬症例から実際の経験症例への展開
- **多職種の視点**: 第3回で社会福祉士・訪問看護師をゲスト招聘
- **PBL形式**: 参加者主体のグループディスカッション

## 2. 研究デザインと理論的枠組み

### 2.1 研究パラダイムと方法論

| 要素 | 内容 |
|---|---|
| **研究パラダイム** | 解釈的パラダイム（interpretive paradigm） |
| **研究目的** | 参加者の学習経験の意味を深く理解する |
| **方法論** | 現象の解釈を重視 |
| **データ収集方法** | 半構造化インタビュー |

### 2.2 理論的枠組み：Kolbの経験学習サイクル

本研究では、Kolb (1984) の経験学習理論を理論的枠組みとして採用する。

#### 経験学習サイクルの4ステージ

```
具体的経験 (Concrete Experience)
        ↓
内省的観察 (Reflective Observation)
        ↓
抽象的概念化 (Abstract Conceptualization)
        ↓
能動的実験 (Active Experimentation)
        ↓
   (サイクルが循環)
```

**学習プロセス**: これらのステージを循環して学びが深められていく

#### 本研究での着目点

1. **CHAT受講前後の「HA実践の具体的経験と内省的観察」**
   - 参加者のHAに関する意識や行動の変化を明らかにする

2. **具体的経験と内省的観察から抽象的概念化へのプロセス**
   - HAの概念理解の深化

3. **抽象的概念化から能動的実験へのつながり**
   - 実践内容の拡張

4. **CHAT受講後の現場での経験学習サイクル**
   - 新たな具体的経験 → 内省的観察 → 抽象的概念化 → 能動的実践

**参考文献**: Kolb (1984) Experiential Learning [19]

## 4. データ収集方法

### 4.1 インタビュー方法

| 項目 | 内容 |
|---|---|
| **形式** | 半構造化インタビュー |
| **実施方法** | オンライン会議システムZoom（同期型） |
| **インタビュアー** | CHATの講師（本論文の筆頭著者） |
| **インタビュー時間** | 1回あたり約60分 |
| **実施期間** | 2023年6月〜10月 |
| **記録方法** | 研究参加者の同意を得た上で録画 |
| **逐語録作成** | 専門業者によって転写 |

### 4.2 インタビュー項目

全11項目のインタビューガイドを作成し、以下の領域について質問を実施：

| 項目番号 | 質問内容 | 探索領域 |
|---|---|---|
| 1 | あなたにとって、HAとはなんだと思いますか？ | **概念理解** |
| 2 | CHAT受講前を振り返ってみて、すでにHAを実践していたと思う症例や出来事はありますか？ | **受講前の実践** |
| 3 | それはどんなHAでしたか？ | **受講前の実践内容** |
| 4 | 受講前と受講後で臨床実践に関して、何か変化したことはありますか？ | **行動変容** |
| 5 | CHAT終了後にHAを実践した症例はありましたか？ | **受講後の実践** |
| 6 | どのような症例でしたか？ | **具体的事例** |
| 7 | どのような課題がありましたか？ | **課題認識** |
| 8 | HAの実践を通して、その症例や家族にどのような変化があったか？ | **実践効果** |
| 9 | 今後、その症例や家族、そして社会に対してさらにどのようなことができると思いますか？ | **将来展望** |
| 10 | 今後、自分のHA実践の質を高めていく上で、研修内容や研修環境など教育面でどのようなことが必要であると思いますか？ | **教育ニーズ** |
| 11 | 今後、小児科医としてどのようなことができると思いますか？ | **職業観の変化** |

**インタビューの特徴**:
- インタビューガイドに即して実施
- 参加者の自由な発言を妨げないよう配慮
- 探索的な質問によって深い洞察を引き出す

## 5. データ分析方法

### 5.1 分析手法：主題分析（Thematic Analysis）

Braun & Clarke (2006) が提示する主題分析の手法を採用

**参考文献**: Braun & Clarke (2006) Using thematic analysis in psychology [20]

### 5.2 分析プロセス

#### ステップ1: セグメント化
- 6名の参加者の半構造化インタビューの逐語録を解析対象
- 逐語録をひとつの意味内容が述べられていると判断したパートでセグメント化

#### ステップ2: コード化
- 各セグメントを短い単語や語句、文でコードを付す

#### ステップ3: サブテーマの作成
- コードと逐語を見返す
- 各コード間を比較
- 類似のものを集めてサブテーマを作成

#### ステップ4: テーマの抽出
- サブテーマ間で比較を行う
- テーマを抽出

#### ステップ5: 精緻化
- コード名、サブテーマ名、テーマ名を見返しながら修正を重ねる

#### ステップ6: カテゴリー分類
- 複数あるテーマをカテゴリーで分類

### 5.3 分析の焦点

以下の3つの側面に焦点を当てて分析：

1. **概念理解の変化**: CHAT受講前後でのHAに関する概念理解の変化
2. **実践内容の変化**: CHAT受講前後での実践内容の変化
3. **新たな視点の獲得**: HAの実践プロセスや実践環境に対する新たな視点

## 7. 研究の質の担保

### 7.1 信頼性（Credibility）

- **三角測定**: 複数の参加者からデータ収集
- **長時間のインタビュー**: 平均68分の深い対話
- **豊富なデータ**: 合計54,202字の逐語録

### 7.2 転移可能性（Transferability）

- **詳細な記述**: 研究参加者の属性を明確に記載
- **文脈の提供**: CHATプログラムの詳細を提示

### 7.3 一貫性（Dependability）

- **明確な分析プロセス**: 主題分析の各ステップを詳細に記載
- **理論的枠組みの使用**: Kolbの経験学習サイクルを一貫して適用

### 7.4 確証性（Confirmability）

- **逐語録の使用**: 参加者の生の声を根拠として提示
- **コードからテーマへの追跡可能性**: 分析プロセスの透明性

## 8. 研究の限界

### 8.1 サンプルサイズ

- **参加者数**: 6名（少数）
- **対応策**: Information power（情報の豊かさ）を重視
  - Malterud et al. (2016) の指摘に基づく [26]
  - 参加者数よりも得られたデータの質を重視

### 8.2 バイアスの可能性

**社会的望ましさバイアス**:
- インタビュアーがCHAT講師であることの影響

**インタビュアーバイアス**:
- 責任研究者がインタビューを実施

**対応策**:
- インタビューガイドの使用
- 参加者の自由な発言を妨げないよう配慮

### 8.3 一般化可能性

- 特定のプログラムに関心を持つ参加者
- 一般的な小児科専攻医と比較すると、能力や意識が高かった可能性
