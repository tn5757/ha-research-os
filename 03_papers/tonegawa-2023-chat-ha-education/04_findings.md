主題分析により、CHAT受講前後のHAに対する認識や実践の変化に関するテーマを7カテゴリに分類した。

## 4.1 概念理解

| | CHAT受講前 | CHAT受講後 |
|---|---|---|
| テーマ | 定義理解の欠如 / 方法理解の欠如 | 定義への理解 / 方法への理解 / 効果への理解 |
| サブテーマ | - | 代弁・エンパワーメント・全人的な関わり / 複合的情報収集・多面多角的な洞察や配慮・複合的な連携 |

- **受講前**: HAという概念や言葉自体を認識していなかった。多職種の視点やHA実践の具体的リソースについても情報を得ていなかった
- **受講後**: HAの定義（代弁、エンパワーメント、全人的な関わり）、方法（複合的情報収集、多面多角的な洞察、複合的な連携）、効果への理解が生まれた

## 4.2 実践内容

| | CHAT受講前 | CHAT受講後 |
|---|---|---|
| テーマ | 多面多角的な情報収集 / 多面多角的な洞察や配慮 / 他者との連携 | 情報収集・洞察や配慮・連携を複合的に組み合わせたアプローチ |
| サブテーマ | 多面的コミュニケーション・他科上級医との連携 / 患者家族に関する洞察や配慮・医療的洞察や配慮 / 多職種連携・家族との連携 | - |

- **受講前**: 情報収集・洞察・連携の各側面いずれか一つに関して、一部のHA手法を使った実践にとどまっていた
- **受講後**: 各側面でバイオサイコソーシャルな視点が加わり手法が拡張。さらに情報収集・洞察・連携を**複合的に組み合わせた一連のアプローチ**が実践されるようになった

## 4.3 実践障壁

| カテゴリ | CHAT受講前サブテーマ | CHAT受講後サブテーマ |
|---|---|---|
| 実践者本人の意識や能力の欠如 | 当事者意識の欠如 / HA想起の欠如 / 多職種への理解や関係性の欠如 | コミュニケーションスキルの欠如 / 多職種への理解や関係性の欠如 / 目標設定の難しさ |
| 患者・家族の状況の難しさ | 非協力的な患者や家族 / 複雑な患者背景 / 医療的な難しさ | 非協力的な患者・家族 / 不完全な医療アクセス / 信頼関係の欠如 / 言語の壁 / プライバシー保護の問題 |
| 多職種の意識や熱意の欠如 | チーム内でのHAへの理解や熱意の差 / 地域連携のしづらさ | チーム内でのHAへの理解や熱意の差 / 地域連携のしづらさ |
| 就労環境上の問題 | 研修の忙しさ / 研修制度 / 研修医という立場 | 研修の忙しさ / 研修制度 / 科や研修医という立場 / ネガティブな雰囲気 |
| 連携システムの問題 | - | 情報アクセス / 連携先の選定 |

- 受講後にサブテーマ数が増加し、考察の具体性が増した（4テーマ11サブテーマ → 5テーマ16サブテーマ）
- 「連携システムの問題」は受講後に新たに出現したテーマ
- 一部の障壁（チーム内の理解差、地域連携のしづらさ等）はCHAT受講単体では取り除かれない

## 4.4 実践促進要素

受講前には言及なし。受講後に3テーマ12サブテーマが出現：

| テーマ | サブテーマ |
|---|---|
| 実践者本人の意識や能力 | 効果的なコミュニケーションへの意識・工夫 / 役割分担の意識 / 他者（連携先主体）への配慮 / HAのより深い理解 / 働く意欲（モチベーション）の管理 / 多職種への理解促進や関係性の構築 |
| 協力的な連携先 | 多職種の共通理解や熱意 / 助け合い精神 |
| 親和性の高い組織環境・地域環境 | 確立された連携システム / 豊富な地域連携先 / 豊富な人材 |

## 4.5 政策提言の想起

受講前には言及なし。受講後に2テーマ7サブテーマ：

| テーマ | サブテーマ |
|---|---|
| 政策課題への注目や提案 | 医療格差の是正 / プライバシー保護に関する法整備 / 第三者機関の設置 / 必要な人材の確保 / 主治医認定制度 |
| システム整備への提案 | 連携システムの整備 / 情報ポータル |

実践障壁の経験から政策提言が想起されるパターン（例：地域連携のしづらさ → 連携システムの整備）が観察された。

## 4.7 HA教育への提案

受講後に1テーマ6サブテーマ：

| サブテーマ | 要点 |
|---|---|
| 社会調整に関するレクチャー | 施設ごとのリソースや連携先の説明が必要 |
| 実地演習 | 座学だけではHA能力は伸びない |
| HAを習得しやすい環境 | 複雑な患者背景がある専門病院での学び |
| 必修化 | 全員が受講すべき |

| 継続的学習機会の欠如 | HAの伸ばし方がわからない |
