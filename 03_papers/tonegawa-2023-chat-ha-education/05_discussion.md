## 5.1 CHATの到達目標の達成とCHATによる認識および行動変容（RQs 1&2）

CHAT受講前後の認識および行動変容を5つの視点で大別：

| 変容の視点 | 経験学習サイクルとの対応 | CHATとの関係 |
|---|---|---|
| ⅰ. 概念理解 | 内省的観察 → 抽象的概念化 | CHATが直接促進 |
| ⅱ. 実践内容の拡張 | 能動的実験 → 具体的経験 | CHAT後の実践で発現 |
| ⅲ. 実践内容の複合化 | 能動的実験 → 具体的経験 | CHAT後の実践で発現 |
| ⅳ. 実践環境への考察の拡張 | 具体的経験 → 内省的観察 | CHAT後のサイクルで発現 |
| ⅴ. HAに関する認識の構造化 | 抽象的概念化 → 能動的実験 | CHAT後のサイクルで発現 |

### ⅰ. 概念理解
- CHATの講義・PBLの組み合わせが理解の深化に有効（Martin 2013と一致）
- CHATの到達目標1〜6の達成を示唆

### ⅱ. 実践内容の拡張 / ⅲ. 実践内容の複合化
- 受講前: 単体の手法で限定的な局面に対処
- 受講後: 多職種連携、社会的配慮を含む複合的アプローチ
- **ミクロレベルのHAに限局**していた点は先行研究（Wright 2005）と一致
- CHATの到達目標1〜6の達成を示唆

### ⅳ. 実践環境への考察の拡張 / ⅴ. HAに関する認識の構造化
- 実践障壁がより具体化（サブテーマ数が11→16に増加）
- 受講前になかった「実践促進要素」が語られるように
- 「政策提言の想起」「理想の医師像への影響」「HA教育への関心」が出現
- **経験学習サイクルがCHAT後も自律的に回転していることを示唆**

### 重要な仮説の生成

> 短期的なプログラムでは、マクロレベルのHAの実践には至らないものの、**メゾ・マクロレベルのHAへの着眼という認識の変容を促すことができる**

## 5.2 今後のHA教育に関する展望（RQ3）

### 5.2.1 CanMEDSコンピテンシーとの乖離

**キー・コンピテンシー1（ミクロレベル）**: CHAT受講により概ね達成可能
**キー・コンピテンシー2（メゾ・マクロレベル）**: 以下の乖離あり

| イネイブリング・コンピテンシー | 達成状況 |
|---|---|
| 2.1 健康の決定要因の特定 | △ 考察には至るが行動には至らない |
| 2.2 継続的な質向上プロセス | × 実践プロセスの改善なし |
| 2.3 地域の健康状態改善への貢献 | × 政策提言の想起のみで実践なし |

→ **「実践プロセスの改善」と「政策提言の実践」** への対処が今後の課題

### 5.2.2 今後目指すべきHA教育への提言

**政策提言の実践に向けて:**
- CHATに加えて政策提言の方法に関する実践的学習の導入
- 米国AAPのAdvocacy GuideやFederal Advocacy Action Networkのような仕組みの本邦版
- 情報やHA実践の取り組みを共有できる共同体の形成

**実践プロセスの改善に向けて:**

| 障壁カテゴリ | 提言 |
|---|---|
| 実践者本人の意識・能力 | コミュニケーション/リーダーシップ講習、目標設定の工夫（「なぜ課題を解決するか」への着眼） |
| 患者・家族の状況 | ロールプレイ/OSCE、多様な背景のケーススタディ、HAホットライン |
| 多職種の意識・熱意 | 小児科医に限らず多職種全体を対象としたプログラムへの拡大 |
| 就労環境 | 同上（組織全体でのHA理解の浸透が必要） |
| 連携システム | 同上 |

**評価ツールについて:**
- より効率的で標準的な評価ツールの提供が必要
- 現存するCanMEDS推奨ツールやポートフォリオは一部の有効性にとどまる
